About the Model

🏗️ Architecture

DenseNet121 with transfer learning: GlobalAveragePooling → Dense(512, ReLU) → Dropout(0.5) → Dense(256, ReLU) → Sigmoid output.

📁 Dataset

Trained on the BreakHis (Breast Cancer Histopathological) dataset consisting of microscopic biopsy images at different magnification levels. The dataset includes both benign and malignant tumor samples, enabling robust learning of tissue-level patterns for accurate classification.

⚙️ Training Strategy

Transfer learning using DenseNet121 backbone. Phase 1: Frozen base for feature extraction. Phase 2: Fine-tuning deeper layers for improved accuracy. Includes data augmentation techniques such as rotation, flipping, and contrast adjustment to improve generalization.

📈 Performance

Achieves ~95% accuracy on the BreakHis test dataset. Balanced performance across benign and malignant classes with strong precision, recall, and F1-score. Optimized using Adam optimizer with early stopping and learning rate scheduling.
